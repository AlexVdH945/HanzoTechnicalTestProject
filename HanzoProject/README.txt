No installation should be required as a virtual environment has been included with all needed packages.

In order to spin up the server. Activate the venv by:

1) .venv/scripts/activate
2) python manage.py runserver

Next in a browser navigate to:

http://127.0.0.1:8000/PDFViewer/FrontPag

This was intended to be the landing page. On this page there will be two buttons. One redirects to a PDF file viewer.
The other will cause the file to be downloaded as "example.pdf"
